## Kachra Seth

This is the project I'd point you to if you wanted to see what I can build. It's a waste management platform with a citizen-facing app and a dashboard for municipal staff.

Some of what went into it:
- bins get tracked using QR codes instead of IoT sensors, which cut deployment costs by around 90%
- collection routes are optimized using a graph traversal algorithm, which brought fuel costs down 30% and made collection noticeably more efficient
- there's an AI piece that looks at a photo of trash and tells you which bin it goes in
- the whole backend is REST APIs I built myself, including a Green Score leaderboard that actual city stakeholders use
